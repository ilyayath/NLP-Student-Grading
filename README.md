# Курсова робота
**Тема:** Розробка системи семантичного оцінювання текстових відповідей студентів українською мовою на основі трансформерних архітектур

## Опис проєкту
Дана робота присвячена автоматизації процесу оцінювання текстових відповідей у навчальних системах. Основний акцент зроблено на використанні методів обробки природної мови (NLP) для визначення смислової близькості між вільною відповіддю студента та еталонною відповіддю викладача.

## Мета та завдання
**Мета роботи:** Розробка програмного комплексу для об’єктивного семантичного аналізу текстів українською мовою, що дозволяє оцінювати відповіді за змістом, а не лише за наявністю ключових слів.

**Завдання проєкту:**
1. Дослідження сучасних трансформерних моделей (BERT, RoBERTa) для генерації семантичних векторів (embeddings).
2. Реалізація алгоритму розрахунку косинусної схожості між векторами текстів.
3. Побудова REST API для інтеграції моделі з зовнішніми інтерфейсами.
4. Забезпечення контейнеризації рішення для подальшого розгортання.

## Технологічний стек
* **Мова програмування:** Python 3.10+
* **Бібліотеки машинного навчання:** PyTorch, Transformers, Sentence-Transformers
* **Обробка природної мови (NLP):** Stanza (для лематизації та морфологічного аналізу)
* **Backend:** FastAPI, Uvicorn
* **Інструменти розробки:** Git, Docker, SQL

## Структура репозиторію
* `data/` — набори даних, еталонні відповіді та тестові вибірки у форматі JSON.
* `src/` — програмна реалізація системи (ядро моделі, обробка тексту, API-ендпоінти).
* `notebooks/` — результати експериментальних досліджень та аналізу точності моделей.
* `docs/` — технічна документація та пояснювальна записка до курсової роботи.
